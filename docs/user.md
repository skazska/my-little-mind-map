# User Guide

This document provides guidance for end-users of the MindMap product. It covers usage and features to help users effectively utilize the product.

## Desktop App

### Status Bar

A status bar is displayed at the bottom of the desktop app window across all views. It shows:

- **Storage path** — the directory where all notes, topics, and indexes are stored on disk.
- **Counts** — current number of notes and topics.
- **App version** — the application version.
